# Todo App using Context API in React

This project is a simple Todo application built using React.js and leveraging the Context API for state management. It allows users to add, update, delete, and mark todos as complete.

## Features

- **Add Todo**: Users can add new todos by entering a task and clicking the "Add" button.
- **Update Todo**: Existing todos can be edited inline by clicking on the todo text and modifying it.
- **Delete Todo**: Users can delete todos by clicking on the delete button associated with each todo.
- **Mark as Complete**: Todos can be marked as complete by clicking on the checkbox next to each todo item.

## Technologies Used

- **React.js**: A JavaScript library for building user interfaces.
- **Context API**: React's Context API is used for state management, providing a centralized place to store and manage todo data.
- **Bootstrap**: Bootstrap is used for styling and layout of the application components.
- **Font Awesome Icons**: Icons from Font Awesome are used for buttons and visual elements.

## Installation

To run this project locally, follow these steps:

1. Clone this repository to your local machine using `git clone`.
2. Navigate to the project directory.
3. Install dependencies using `npm install`.
4. Run the development server using `npm start`.

## Project Structure

The project structure is organized as follows:

- **`src/`**: Contains the source code for the application.
  - **`Components/`**: Contains React components used in the application.
    - `AddTodo.jsx`: Component for adding new todos.
    - `TodoForm.jsx`: Component for displaying and managing todo items.
  - **`Context/`**: Contains files related to context management.
    - `TodoContext.jsx`: Defines the TodoContext and TodoProvider for managing todo data.
  - **`App.jsx`**: Main component rendering the Todo App.
  - **`index.css`**: Stylesheet for the application.
  - **`index.js`**: Entry point of the application.

## Usage

Once the application is running locally, you can access it in your web browser. Add new todos, update existing ones, mark them as complete, and delete them as needed.
